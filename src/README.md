## ユーザーマニュアル

### 1. ビルド方法

プログラムをビルドするには、以下のコマンドを実行してください。

```bash
g++ -std=c++17 main.cpp -o main
```

### 2. 実行方法
```bash
./main
```

### 3/ 操作画面の注意

#### ランドマーク説明

G：ゴール


### 4. ルール
ガソリン容量が0になる前にゴール(G)を目指します。
車が進んだ距離分ガソリンが減ります。
ガスステーションでガソリンを補給することができます。
車の状態や地図の状態を標準出力で表示しています。
